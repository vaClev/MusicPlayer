Чтобы приложение JAVA FX запускалось у пользователей:
Нужно создать пакет установщика.

-----
Обходное решение:
1. Установаить Open JDK моей версии 21.0.9
2. Распаковать openjfx моей версии 21.0.9
3. Подкинуть JAR файл - который получается запуском "mvn package" (в Intelliji IDEA)
4. Запускать start.bat

--- 
Для Linux и MacOS - не исследовал, но точно нужно будет устанавливать Open JDK и openjfx
