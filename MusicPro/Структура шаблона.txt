               ▼

Как это работает вместе:
Последовательность запуска:
Launcher.main() → HelloApplication.main()

Application.launch() → создает JavaFX Application Thread

HelloApplication.start(Stage) → загружает FXML

FXMLLoader создает UI элементы из FXML

FXMLLoader создает экземпляр HelloController

Связывание: поля с @FXML заполняются ссылками на UI элементы

initialize() (неявно) - вызывается после связывания всех @FXML полей

Stage.show() - отображается окно
